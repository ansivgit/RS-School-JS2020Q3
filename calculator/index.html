• Для расчета значения квадратного корня из числа необходимо ввести число и нажать кнопку корня. Аналогично реализована кнопка 1/x;
  • Для смены знака числа необходимо сперва ввести число, затем нажать ±;
  • Ввод десятичной дроби меньше 1 допускается как в виде .xxx, так и в виде 0.xxx;
  • Для извлечения корней более 2 степени используется выражение операнд xn показатель степени в формате десятичной дроби;
  • В случае введения подряд нескольких операндов, операции над ними осуществляются последовательно, каждая последующая операция равносильна знаку =, т.е. 7 + 2 * 3 === 9 * 3 => 27.
